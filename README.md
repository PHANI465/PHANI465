- GPT-4o-mini as judge — 25× cheaper, ~$0.87/run vs ~$21.75 with GPT-4o

**Tech:** Python · LangChain · RAGAS · OpenAI GPT-4o · Pinecone · GitHub Actions · React · Vite · SQLite
